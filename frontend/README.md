# 영화 기록 앱 (Movie Tracker)

React Native/Expo로 제작된 영화 기록 및 관리 앱입니다.

## 설치 방법

1. 의존성 설치:
\`\`\`bash
npm install
\`\`\`

2. 앱 실행:
\`\`\`bash
# iOS
npm run ios

# Android
npm run android

# 웹
npm run web
\`\`\`

## 주요 기능

- 영화 기록 및 관리
- 별점 및 리뷰 작성
- 통계 및 분석
- 태그 시스템
- 컬렉션 관리

## 기술 스택

- React Native
- Expo
- React Navigation
- TypeScript
